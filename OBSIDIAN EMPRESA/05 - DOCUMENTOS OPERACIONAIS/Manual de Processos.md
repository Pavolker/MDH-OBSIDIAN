# Manual de Processos

Documenta os fluxos de trabalho, procedimentos, responsáveis e padrões operacionais da empresa.

## Estrutura Recomendada

| Elemento | Descrição |
|----------|-----------|
| **Fluxograma** | Representação visual do processo |
| **Descrição** | Passo a passo textual |
| **Responsável** | Dono do processo |
| **Entradas** | Documentos, dados, insumos |
| **Saídas** | Entregas, resultados |
| **Indicadores** | Tempo, qualidade, custo |
| **Exceções** | Tratamento de erros |
