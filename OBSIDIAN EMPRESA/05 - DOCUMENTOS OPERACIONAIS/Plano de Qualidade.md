# Plano de Qualidade

Define os padrões de qualidade, os processos de auditoria, o controle e a rastreabilidade dos produtos e serviços.

## Componentes

| Componente | Descrição |
|------------|-----------|
| **Padrões** | Especificações, critérios de aceitação |
| **Controle** | Inspeção, teste, medição |
| **Auditoria** | Interna, externa, certificação |
| **Não Conformidade** | Identificação, correção, prevenção |
| **Rastreabilidade** | Lotes, versões, histórico |
| **Melhoria** | Ação corretiva, preventiva, PDCA |
