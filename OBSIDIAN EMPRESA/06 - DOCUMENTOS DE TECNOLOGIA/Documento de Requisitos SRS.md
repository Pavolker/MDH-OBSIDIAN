# Documento de Requisitos (SRS)

Software Requirements Specification — especifica funcionalidades, regras de negócio e requisitos técnicos do sistema.

## Estrutura

| Seção | Descrição |
|-------|-----------|
| **Introdução** | Propósito, escopo, definições |
| **Descrição Geral** | Perspectiva, usuários, premissas |
| **Requisitos Funcionais** | Funcionalidades, casos de uso |
| **Requisitos Não Funcionais** | Performance, segurança, usabilidade |
| **Regras de Negócio** | Lógicas, validações, restrições |
| **Interface** | Telas, APIs, relatórios |
