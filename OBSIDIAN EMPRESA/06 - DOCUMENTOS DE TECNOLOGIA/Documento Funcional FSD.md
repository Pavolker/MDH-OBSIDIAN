# Documento Funcional (FSD)

Functional Specification Document — descreve o comportamento do sistema, telas, operações e fluxos de interação.

## Estrutura

| Seção | Descrição |
|-------|-----------|
| **Telas** | Layout, campos, ações, navegação |
| **Fluxos** | Sequência de telas e operações |
| **Regras** | Validações, cálculos, estados |
| **Mensagens** | Erros, confirmações, alertas |
| **Permissões** | Quem pode fazer o quê |
| **Relatórios** | Estrutura, filtros, exportação |
