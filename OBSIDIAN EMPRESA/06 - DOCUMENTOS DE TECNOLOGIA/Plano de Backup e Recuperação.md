# Plano de Backup e Recuperação

Define as políticas de redundância, frequência de backup, procedimentos de restauração e garantia de continuidade operacional.

## Componentes

| Componente | Descrição |
|------------|-----------|
| **Política** | O que, quando, como fazer backup |
| **Frequência** | Diária, semanal, tempo real |
| **Retenção** | Dias, meses, anos |
| **Armazenamento** | Local, cloud, híbrido |
| **Teste** | Procedimento de validação |
| **RTO / RPO** | Tempo de recuperação e perda aceitável |
