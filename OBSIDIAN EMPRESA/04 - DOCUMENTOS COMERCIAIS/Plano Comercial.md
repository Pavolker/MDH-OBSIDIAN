# Plano Comercial

Contém as metas de vendas, a definição do cliente ideal (ICP), a abordagem comercial, os canais de venda e os indicadores de performance.

## Componentes

| Componente | Descrição |
|------------|-----------|
| **Metas** | Receita, unidades, novos clientes |
| **ICP** | Perfil do cliente ideal |
| **Abordagem** | Consultiva, transacional, inside sales |
| **Canais** | Direto, parceiros, marketplace |
| **CRM** | Gestão de leads, pipeline, forecast |
| **Indicadores** | Conversão, ticket médio, ciclo |
