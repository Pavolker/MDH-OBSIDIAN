# Plano de Continuidade de Negócios (BCP)

Define procedimentos de resposta a crises, contingência operacional e garantia de continuidade do negócio.

## Componentes

| Componente | Descrição |
|------------|-----------|
| **Análise de Impacto** | Processos críticos, MTD, MBCO |
| **Estratégias** | Contingência, redundância, terceirização |
| **Equipe** | Responsáveis, contatos, comitê |
| **Comunicação** | Stakeholders, clientes, imprensa |
| **Ativação** | Gatilhos, níveis de crise |
| **Retomada** | Passos para normalização |
