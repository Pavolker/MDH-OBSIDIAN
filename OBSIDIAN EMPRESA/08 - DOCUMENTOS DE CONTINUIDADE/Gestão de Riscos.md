# Gestão de Riscos

Mapeia riscos financeiros, operacionais, tecnológicos, jurídicos e reputacionais, com planos de mitigação.

## Etapas

| Etapa | Descrição |
|-------|-----------|
| **Identificação** | Categorias, fontes, cenários |
| **Análise** | Probabilidade, impacto |
| **Avaliação** | Criticidade, priorização |
| **Tratamento** | Mitigação, transferência, aceitação |
| **Monitoramento** | Indicadores, revisão periódica |
| **Comunicação** | Reporte, escalonamento |
