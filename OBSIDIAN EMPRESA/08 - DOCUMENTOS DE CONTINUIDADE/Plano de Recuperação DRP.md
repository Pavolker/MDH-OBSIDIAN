# Plano de Recuperação de Desastres (DRP)

Voltado para a recuperação de infraestrutura, dados e sistemas críticos após um desastre.

## Componentes

| Componente | Descrição |
|------------|-----------|
| **Sistemas Críticos** | Prioridade de recuperação |
| **RTO** | Recovery Time Objective |
| **RPO** | Recovery Point Objective |
| **Infraestrutura** | Servidores, rede, cloud |
| **Dados** | Backup, replicação, restauração |
| **Teste** | Simulação, validação periódica |
