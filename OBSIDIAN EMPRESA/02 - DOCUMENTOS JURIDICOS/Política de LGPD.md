# Política de LGPD e Privacidade

Documento necessário para empresas que tratam dados pessoais. Define consentimento, segurança, retenção e uso de dados em conformidade com a Lei Geral de Proteção de Dados.

## Direitos dos Titulares

| Direito | Descrição |
|---------|-----------|
| **Confirmação** | Saber se tratamos seus dados |
| **Acesso** | Acessar dados armazenados |
| **Correção** | Corrigir dados incompletos |
| **Exclusão** | Solicitar eliminação |
| **Oposição** | Recusar tratamento |
| **Portabilidade** | Transferir dados |
