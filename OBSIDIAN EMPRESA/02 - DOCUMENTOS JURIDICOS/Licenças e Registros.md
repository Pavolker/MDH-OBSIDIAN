# Licenças e Registros

Conjunto de autorizações legais necessárias para a operação regular da empresa.

## Principais

| Documento | Órgão |
|-----------|-------|
| **CNPJ** | Receita Federal |
| **Inscrição Estadual** | SEFAZ |
| **Inscrição Municipal** | Prefeitura |
| **Alvará de Funcionamento** | Prefeitura |
| **Licença Sanitária** | Vigilância Sanitária |
| **Licença Ambiental** | Órgão ambiental |
| **Autorizações Setoriais** | ANVISA, ANATEL, etc. |
